BUILDVERSE PLAYER
Prototype Game Launcher

Status: Prototype (v0.1)

The Buildverse Player is an early prototype built in Java (.jar). It allows users to browse games and launch them directly in the Buildverse Studio.

🧠 How it works:

  • Select a game from the list
  • Click PLAY
  • The game opens in Buildverse Studio (Python)
