# BUILDVERSE
### Bringing back the 2004 Roblox experience!

---

## 🕹 Project Overview
Buildverse is a fan-made recreation of the classic **2004 Roblox engine**. Our goal is to bring back the early physics, the original brick-building mechanics, and the simple aesthetic that started it all.

### Features:
*   **Original 2004 Physics:** Experience the classic blocky movements.
*   **Legacy Studio Support:** Build like it's 2004.
*   **Retro UI:** Simple and clean interface.
